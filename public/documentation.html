Movie API

This API is supposed to provide Information about diffrent Movies, directors and genres. The user will be able to sign up, update their Information and create a List of their favorite movies.

Api Endpoints Table

Name (Url) Http Method Parameter Request Format Response Format
http://localhost:8080/movies GET none none Json Object with a list of Movies
http://localhost:8080/movies/[Title] GET title none Json Object with Data about a single Movie
http://localhost:8080/movies/[Title]/genres GET title none Json Object with Description about a Genre of specified Movie
http://localhost:8080/movies/[Title]/director GET title none Json Object with Description about the Dictor of the specified Movie
http://localhost:8080/users POST none Json Object with Users Login Info Json Object that returns the User Info with added ID
http://localhost:8080/users/[ID]/[Username] PUT id,username none Message that confirms that the Username was changed
http://localhost:8080/users/[ID]/favorite POST id JSON Object that either Contains the ID or Title of a Movie from the /movies List Message that confirms that the Movie was added to Favorites
http://localhost:8080/users/[ID]/favorite/[Title] DELETE id,title none Message that confirms that the Movie was deletet to Favorites
http://localhost:8080/users/[ID] DELETE id none Message that confirms that the Users Email has been Removed formt he Userslist
